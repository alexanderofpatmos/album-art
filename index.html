ABOUT THIS WEBSITE Background Development Contact

Hello, my name is Alexander. I like album covers.

I believe cover artwork is a very essential part of releasing a work of music, because it may influence how the listener perceives the music they hear. I created this website to catalog some of my personal favorite album covers.

The code for this website was generated with artificial intelligence, because I personally don't know the first thing about coding. I laid down the foundation with ChatGPT 4.0 but quickly realized GitHub Copilot is superior and more efficient. Claude 3.5 was also formidable, but user usage is extremely limited.

My text editor of choice is Virtual Studio Code.

This website is being hosted for free on GitHub Pages.

Contact me at: alexander@example.com
